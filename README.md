# ikun Loader
### 为ikun打造的控制台加载动画


## Demo Preview
![demo](demo.gif)

## Install
```shell
npm i ikun-loader

```

## Usage
```javascript
import { ikunDance, ikunBasketball } from 'ikun-loader';

const ikun = ikunDance('loading');
ikun.succeed('finished');
// OR
ikun.failed('failed');

```
